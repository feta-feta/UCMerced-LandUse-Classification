# Aerial Scene Classification (UC Merced Land Use Dataset)

This repository contains a Deep Learning project focused on multi-class image classification for aerial and satellite imagery. The objective is to accurately classify land-use scenes into 21 distinct categories using PyTorch and Convolutional Neural Networks (CNNs).

*This project was developed as part of the Deep Learning course at the University of Tehran, under the supervision of Dr. Ahmad Kalhor.*

## Project Overview
- **Dataset:** UC Merced Land Use Dataset ([link](https://www.kaggle.com/datasets/abdulhasibuddin/uc-merced-land-use-dataset))
- **Task:** Multi-class Image Classification.
- **Frameworks & Libraries:** PyTorch, Torchvision, Timm, Scikit-learn, Pandas.



##  Highlights
- Utilizes hardware acceleration (CUDA/GPU) for efficient training.
- Preserves class balance during data splitting using stratified techniques.
- Evaluates model performance using comprehensive metrics like confusion matrix and classification reports.
